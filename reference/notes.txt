Section 2 - The Basics 

ng serve - index.html (the single page) - <app-root> - main.ts - app.module.ts - [AppComponent] (Segala yang berekstensi component) - component.html
Setiap angular page terdiri atas komponen - komponen.
Component is just a class under root component (app).
selector: html-tag

Decorator : Component(). 
Module : Bundling all the things.
Angular harus dikasih tau ada component apa aja melalui module.
Template in the component is a must.
Template cuma satu, styles bisa reference lebih dari satu.
Data Binding : Communication between ts file and html file.
Square bracket di angular berarti property binding : [disabled]="!allowNewServer"
$event is a super variable name you can use when using event binding.
