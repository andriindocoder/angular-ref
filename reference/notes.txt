ng serve - index.html (the single page) - <app-root> - main.js - app.module.ts - [AppComponent] (Segala yang berekstensi component) - component.html

Component is just a class under root component (app).
Decorator : Component(). 
Module : Bundling all the things.
Angular harus dikasih tau ada component apa aja melalui module.
